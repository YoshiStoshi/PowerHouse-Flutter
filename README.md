# Projeto Eletiva Dispositivos Moveis

Alunos: Rodrigo de Azevedo Junior 2840482421044 e Davi Sousa Cirilo 2840482421066

## Breve Apresentacao

O aplicativo escolhido para ser desenvolvido para esse trabalho foi um aplicativo para a academia Power House Fitness Gym.

Nele, visamos deixá-lo o mais parecido possível com a identidade da academia.

Valores apresentados são fictícios, porém as outras funcionalidades, como:

- Endereço;
- Número de telefone;
- Aulas de Jiu-jitsu;

São informações reais correspondentes ao que foi passado pela academia.
